# PointMax

クレジットカード・ポイントカードからの還元と交換ルートを最適化する、ローカル動作のPWAです。

## 機能

- **計算**: 店舗・金額・目標通貨を選ぶと、保有カード別に最終取得量と最適な交換ルートを比較
- **二重取り**: クレカ決済とポイントカード提示を合算
- **交換ルートグラフ**: 通貨ノード間のレートを編集可能な図で表示（@xyflow/react）
- **シードバージョニング**: サンプルデータを更新するたびユーザーに通知し、既存編集を保護したまま差分のみマージ
- **JSONエクスポート/インポート**: 端末間でデータを持ち回り
- **PWA**: スマホのホーム画面に追加してアプリのように使える

## ローカル開発

```bash
npm install
npm run dev      # http://localhost:5173
npm run test     # Vitest
npm run build    # 本番ビルド
```

## デプロイ

`main` ブランチに push すると GitHub Actions が自動でビルド & GitHub Pages にデプロイされます。

## 注意

- データは `localStorage` に保存（端末/ブラウザごとに別）
- 還元率・交換レートは公式情報に基づく概算。実利用前に各社公式で要確認
- 個人情報は含まれない設計（カード名・利用店舗のみ）
